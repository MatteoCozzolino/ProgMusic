# Progetto di Programmazione per la musica

# Sonorizzazione di un dataset

Il progetto consiste nell'elaborazione di un dataset e nella riproduzione di una melodia in cui ad ogni nota è associato un elemento del dataset stesso. Il dataset in esame è costituito dal numero di arresti che la polizia della città di New York ha compiuto nel corso dell'anno 2020. 

# Generazione melodia

Eseguire *Controller.Launcher*. Il programma offrirà all'utente la possibilità di effettuare delle personalizzazioni riguardati la melodia come il ritmo, il volume e il pan. Alla conclusione della riproduzione della melodia il programma darà all'utente la possibilità di generare un'altra melodia con differenti parametri.
